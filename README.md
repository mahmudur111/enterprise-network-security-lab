# Enterprise Network & Security Lab

## Overview

A hands-on enterprise networking and Linux administration laboratory built on Ubuntu Server using VMware Workstation.

The objective of this project is to develop practical skills in Linux system administration, TCP/IP networking, network security, service deployment, troubleshooting, and automation while building a professional GitHub portfolio.

---

## Lab Environment

- Ubuntu Server 26.04 LTS
- VMware Workstation
- Git & GitHub
- Apache2
- Bind9 DNS
- ISC DHCP Server
- OpenSSH Server
- UFW Firewall
- Wireshark
- tcpdump
- Bash
- Python (coming soon)
