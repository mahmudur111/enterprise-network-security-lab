# Enterprise Network & Security Lab

## Project Overview

This project is designed to develop practical Linux administration, networking, security, monitoring, and automation skills using Ubuntu Server and VMware.

## Objectives

- Learn Linux system administration
- Practice TCP/IP networking
- Implement DNS and DHCP services
- Perform network troubleshooting
- Develop Python-based automation tools
- Build a professional GitHub portfolio

## Technologies

- Ubuntu Server 26.04
- VMware Workstation
- Git & GitHub
- Python
- SSH
- TCP/IP
- DNS
- DHCP
- Nmap
- Wireshark
